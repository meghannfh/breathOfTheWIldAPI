Welcome to The Encyclopedia of Hyrule

To start using this encyclopedia begin typing in the search bar. Items stored in the encyclopedia that start with the same letter/letters you have typed will populate in a list below the search bar for you to choose from.

Alternatively, you can loop through each category(food creatures, non-food creatures, weapons, materials, and treasure) alphabetically by first clicking a category icon. After clicking, the first item's image and information will appear and from there you can use the right or left arrow to loop either backwards or forwards through the list to show the next item.
